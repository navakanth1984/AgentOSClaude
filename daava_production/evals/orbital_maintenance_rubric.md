# Evaluation Rubric: Orbital Maintenance Judge

Use this rubric to evaluate if an AI-generated prompt adheres to the **Orbital Maintenance** managed context.

## 1. Visual DNA Fidelity (Weight: 40%)
- [ ] **Suit Color**: Does it mention Orange or `#E67E22`?
- [ ] **Harness**: Does it mention Deep Blue, Midnight Blue, or `#2C3E50`?
- [ ] **Helmet**: Does it mention Yellow or `#F1C40F`?
- [ ] **Weathering**: Is the equipment described as soot-covered, industrial, or weathered?

## 2. Technical Precision (Weight: 30%)
- [ ] **Lens**: Is a **14mm** ultra-wide lens explicitly specified?
- [ ] **Perspective**: Is a **Worm's-Eye View** or **Extreme Low-Angle** specified?
- [ ] **Lighting**: Does it mention the **Warm Golden Rim Light** (`#F39C12`)?

## 3. Atmospheric Consistency (Weight: 20%)
- [ ] **Environment**: Is it set against a gas giant, lattice truss, or in a vacuum?
- [ ] **Details**: Are dust particles or glowing cyan indicators mentioned?

## 4. Constraint Adherence (Weight: 10%)
- [ ] **Negatives**: Does it avoid forbidden lenses (35mm, 50mm) and clean aesthetics?

---
**Scoring:**
- **Pass (High Fidelity)**: 90% - 100%
- **Pass (Acceptable)**: 80% - 89% (Within Error Budget)
- **Fail**: < 80%
