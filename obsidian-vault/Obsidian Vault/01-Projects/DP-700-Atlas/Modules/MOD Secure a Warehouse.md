# Secure a Warehouse

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Implement a data warehouse / Admin & governance*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/secure-data-warehouse-in-microsoft-fabric/

## Learning objectives
- Explain warehouse security concepts
- Implement dynamic data masking
- Configure row-level security
- Implement column-level security
- Configure granular T-SQL permissions

## Key concepts
DDM masks at query time (default/email/partial/random masks). RLS = security predicate function + security policy. CLS = GRANT SELECT on column list. GRANT/DENY/REVOKE object-level permissions.
