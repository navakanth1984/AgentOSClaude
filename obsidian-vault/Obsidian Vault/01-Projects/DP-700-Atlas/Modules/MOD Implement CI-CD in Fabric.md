# Implement CI-CD in Fabric

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Manage a Fabric environment*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/implement-cicd-in-fabric/

## Learning objectives
- Define CI/CD and how Fabric implements it
- Implement version control and Git integration
- Use deployment pipelines
- Automate CI/CD with Fabric APIs

## Key concepts
Git integration (Azure DevOps/GitHub) at workspace level: connect, commit, update, branch. Deployment pipelines: dev→test→prod stages, deployment rules, variable libraries. fabric-cicd / REST APIs for automation. Database projects for warehouse schema.
