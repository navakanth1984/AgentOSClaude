# Use Eventstream

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Implement Real-Time Intelligence*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/explore-event-streams-microsoft-fabric/

## Learning objectives
- Explain Eventstream components
- Set up sources and destinations
- Capture, transform, and route data

## Key concepts
Sources: Event Hubs, IoT Hub, Kafka, CDC, sample data. Destinations: Eventhouse, Lakehouse, Derived stream, Activator. Transformations: filter, manage fields, aggregate, group by with **windowing** (tumbling/hopping/sliding/session), union, expand, join.
