## Learning objectives
- Create an Eventhouse
- Write effective KQL queries

## Key concepts
Eventhouse contains KQL databases; OneLake availability toggle. KQL: where/summarize/bin()/join/render; query best practices (filter early, limit columns). Materialized views for pre-aggregation; stored functions for reuse; update policies.
