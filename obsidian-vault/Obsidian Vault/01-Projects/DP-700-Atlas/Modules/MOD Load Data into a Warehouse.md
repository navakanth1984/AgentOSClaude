# Load Data into a Warehouse

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Implement a data warehouse*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/load-data-into-microsoft-fabric-data-warehouse/

## Learning objectives
- Compare data load strategies
- Build a pipeline to load a warehouse
- Load with T-SQL
- Load/transform with Dataflow Gen2

## Key concepts
Full vs incremental loads; staging tables. COPY INTO from ADLS/Blob, CREATE TABLE AS SELECT (CTAS), INSERT...SELECT, cross-warehouse ingestion. SCD handling. Pipeline Copy activity vs T-SQL vs Dataflow trade-offs.
