# Secure Data Access in Fabric

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Manage a Fabric environment / Admin & governance*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/secure-data-access-in-fabric/

## Learning objectives
- Describe the Fabric permissions model
- Configure workspace and item permissions
- Apply granular permissions

## Key concepts
Multi-layer model: tenant → capacity → workspace (Admin/Member/Contributor/Viewer) → item permissions (share, build) → granular (RLS/CLS/OneLake data access roles/folder-file level). Principle: most restrictive wins.
