# Get Started with Real-Time Intelligence

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Implement Real-Time Intelligence*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/get-started-kusto-fabric/

## Learning objectives
- Describe real-time analytics, streams and events
- Understand RTI components
- Ingest and transform data in motion
- Store and query KQL databases
- Visualize with Real-Time Dashboards
- Automate with Activator

## Key concepts
RTI stack: Real-Time Hub → Eventstream → Eventhouse (KQL DB) → Real-Time Dashboard → Activator. Streams vs batch; events; KQL basics (take, where, summarize, render).
