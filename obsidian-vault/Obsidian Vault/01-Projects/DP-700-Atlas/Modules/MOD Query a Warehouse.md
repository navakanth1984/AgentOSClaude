# Query a Warehouse

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Implement a data warehouse*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/query-data-warehouse-microsoft-fabric/

## Learning objectives
- Write T-SQL in the SQL query editor
- Build no-code queries in the visual editor
- Connect via SSMS with Entra ID auth

## Key concepts
SQL query editor in Fabric portal; visual query editor (Power Query-style diagram); external tools (SSMS, ADS) via SQL connection string + Microsoft Entra ID. Aggregations, JOINs, ranking functions for analytics.
