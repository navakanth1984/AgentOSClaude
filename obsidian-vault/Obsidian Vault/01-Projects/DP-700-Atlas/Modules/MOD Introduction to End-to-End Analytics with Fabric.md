# Introduction to End-to-End Analytics with Fabric

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Implement a Lakehouse / Get started with Fabric*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/introduction-end-analytics-use-microsoft-fabric/

## Learning objectives
- Identify the capabilities of Microsoft Fabric
- Implement Fabric to meet enterprise analytics needs
- Describe AI capabilities: Copilot, data agents, Fabric IQ

## Key concepts
Fabric = SaaS analytics platform unifying Data Factory, Data Engineering, Data Warehouse, Real-Time Intelligence, Data Science and Power BI on [[OneLake]]. One copy of data, many engines. Workspaces, capacities, admin portal enablement.
