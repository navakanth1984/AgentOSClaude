## Learning objectives
- Apply monitoring concepts to Fabric

## Key concepts
Monitoring Hub aggregates run history for pipelines, dataflows, notebooks, semantic model refreshes, eventstreams. Filter by item type/status/submitter. Activator for alerting on conditions. Capacity Metrics app for CU health.
