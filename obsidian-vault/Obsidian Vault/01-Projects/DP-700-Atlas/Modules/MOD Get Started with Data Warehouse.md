# Get Started with Data Warehouse

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Implement a data warehouse*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/get-started-data-warehouse/

## Learning objectives
- Describe warehouse & dimensional modeling fundamentals
- Create tables and load data
- Query with T-SQL and visual editor
- Model for reporting
- Secure and monitor a warehouse

## Key concepts
Full T-SQL read/write warehouse over OneLake (Delta under the hood). Star schema: fact + dimension tables, surrogate keys, SCDs. Cross-database queries to lakehouse SQL endpoints. Default semantic model + Direct Lake.
