# Monitor a Warehouse

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Implement a data warehouse*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/monitor-fabric-data-warehouse/

## Learning objectives
- Monitor capacity usage with the Capacity Metrics app
- Monitor current activity with DMVs
- Analyze query trends with Query Insights

## Key concepts
Capacity Metrics app: CU consumption, smoothing/throttling. DMVs: sys.dm_exec_connections / _sessions / _requests for live activity. Query Insights views: exec_requests_history, frequently/long running queries.
