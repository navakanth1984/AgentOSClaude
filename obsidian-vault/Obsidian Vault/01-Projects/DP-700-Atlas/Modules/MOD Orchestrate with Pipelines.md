# Orchestrate with Pipelines

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Ingest data / Implement a Lakehouse*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/use-data-factory-pipelines-fabric/

## Learning objectives
- Describe pipeline capabilities
- Use the Copy Data activity
- Create pipelines from templates
- Run and monitor pipelines

## Key concepts
Pipelines = activities (Copy Data, Notebook, Dataflow, stored proc) + control flow (ForEach, If, vars), parameters & dynamic expressions, schedules & event triggers. Copy Data for raw lift; chain notebooks/dataflows for transformation. Monitor in run history / Monitoring hub.
