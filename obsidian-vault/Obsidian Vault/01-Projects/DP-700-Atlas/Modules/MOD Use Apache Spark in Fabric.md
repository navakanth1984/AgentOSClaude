# Use Apache Spark in Fabric

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Implement a Lakehouse / Ingest data*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/use-apache-spark-work-files-lakehouse/

## Learning objectives
- Configure Spark in a Fabric workspace
- Choose between Spark notebooks and Spark jobs
- Use DataFrames to analyze and transform data
- Use Spark SQL on tables and views
- Visualize data in a notebook

## Key concepts
Spark pools (starter vs custom), environments, runtime version, high-concurrency mode. Notebooks for interactive work; Spark job definitions for batch. DataFrame API (read/select/filter/groupBy/write), Spark SQL over managed tables, matplotlib/seaborn for charts.
