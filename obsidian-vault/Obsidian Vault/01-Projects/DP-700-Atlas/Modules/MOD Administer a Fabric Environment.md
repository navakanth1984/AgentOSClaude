# Administer a Fabric Environment

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Manage a Fabric environment / Admin & governance*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/administer-fabric/

## Learning objectives
- Describe Fabric admin tasks
- Navigate the admin center
- Manage user access
- Govern data in Fabric

## Key concepts
Admin portal: tenant/capacity/workspace settings, domains. Governance: endorsement (promote/certify), sensitivity labels, scanner APIs, **audit logs** via Purview. Delegated admin settings.
