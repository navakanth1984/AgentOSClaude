# Get Started with Lakehouses

Up: [[DP-700 Atlas — MOC]]  ·  Learning path: *Implement a Lakehouse*

> MS Learn module: https://learn.microsoft.com/en-us/training/modules/get-started-lakehouses/

## Learning objectives
- Describe lakehouse features and capabilities
- Create a lakehouse
- Ingest and transform data in a lakehouse
- Query lakehouse data with SQL and Spark

## Key concepts
Lakehouse = data lake flexibility + warehouse analytics over Delta tables. Tables vs Files areas, SQL analytics endpoint (read-only T-SQL), default semantic model. Ingest via upload, pipelines, dataflows, shortcuts.
